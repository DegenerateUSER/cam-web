⚙ SERVER SETTINGS

Enter your go2rtc server URL. When deploying remotely, use your ngrok/Cloudflare tunnel HTTPS URL.

Examples:
Local: http://localhost:1984
Ngrok: https://abc123.ngrok-free.app
Cloudflare: https://cam.yourdomain.com

You can also set via URL: ?server=https://your-url
